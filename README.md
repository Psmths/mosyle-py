# mosyle-py
## INTRODUCTION



## USAGE

Make sure to set ```token``` in *device.py* to your Mosyle API token first.

```
python devices.py --command [command] --device [device]
```

Currently supported ```--command``` arguments:

* shutdown
* restart
* info
